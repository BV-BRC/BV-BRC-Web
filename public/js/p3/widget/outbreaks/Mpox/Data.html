Monkeypox outbreak FASTA, MSA and phylogenetic tree files from 2024 and 2025 available from the BV-BRC FTP Site .

These files contain the data used to generate the phylogenetic trees provided on the Phylogenetics tab on the MPOX Outbreak page.

Select genome sequences were downloaded from NCBI and BV-BRC to cover all major clades.

Sequences were then aligned using MAFFT. Multiple sequence alignment columns with more than 50% gaps were deleted. A minimal evolution phylogenetic tree was then inferred with the FastME program based on maximum likelihood pairwise distances calculated by TREE-PUZZLE using the GTR model and 100 bootstrap re-samplings. Custom scripts were used to add meta-data to the resulting tree. Clade information is based on NEXSTRAIN monkeypox virus phylogenetics.
